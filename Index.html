Tags de Texto:

As tags de texto como o proprio nome diz são tags utilizadas para exibir textos, porem cada uma delas executa essa função de forma diferente. A primeira Tag aprendida é a "p" ela é a tag parágrafo, onde todo texto escrito é inserido em um único parágrafo.
Também temos as tags h1-h6 que podem ser entendidas como títulos onde a numeração indica o tamanho da fonte,
Por ultimo nas Tags de Texto possuimos varias tags que são como "Atributos" podendo editar o texto ou palavras dentro do texto, sendo elas u sublinhado, i italico, strong negrito e mark marca texto.
